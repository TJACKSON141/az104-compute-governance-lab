# Azure Compute & Governance Lab (AZ-104)

## Overview
This project demonstrates how to deploy and govern Azure compute resources using enterprise-grade practices: RBAC, Key Vault, Azure Policy, NSGs, and Cost Management.

It aligns with AZ-104 objectives for:
- Virtual Machines
- Identity & Access Management
- Governance & Compliance
- Cost Management
